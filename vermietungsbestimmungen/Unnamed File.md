# 

**Typ Einträge:** Vermietungsbestimmungen

---

Zur Dauer des Wohnens oder einer spezifischen Bezeichnung wie Bleiberecht oder Wohnrecht finden sich auf der Webseite keine Angaben. Es wird lediglich beschrieben, dass die Vermietungs-Bestimmungen in einem Reglement festgelegt sind, das vom Stiftungsrat verabschiedet und vom Stadtrat genehmigt wurde.

Zu den Gründen für einen Ausschluss aus der Genossenschaft oder für eine Kündigung der Wohnung sowie zu einem möglichen Ersatzangebot finden sich auf der Webseite keine Informationen.
